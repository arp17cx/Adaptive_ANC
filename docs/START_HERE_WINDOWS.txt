═══════════════════════════════════════════════════════════
  自适应ANC系统 - Windows快速开始 (3步完成)
═══════════════════════════════════════════════════════════

【步骤1】安装GCC编译器 (只需一次)
────────────────────────────────────
1. 下载 TDM-GCC:
   https://jmeubank.github.io/tdm-gcc/
   
2. 运行安装程序
   
3. 安装时勾选: "Add to PATH"
   
4. 重启电脑

【步骤2】解压所有文件
────────────────────────────────────
将下载的所有文件解压到同一个文件夹，例如:
C:\ANC_System\

确保以下文件都在:
  ✓ run.bat           (最重要!)
  ✓ build.bat
  ✓ clean.bat
  ✓ main.c
  ✓ 其他所有 .c 和 .h 文件

【步骤3】运行
────────────────────────────────────
双击: run.bat

就这么简单！

═══════════════════════════════════════════════════════════
  输出文件
═══════════════════════════════════════════════════════════

运行完成后会生成:
  ✓ anc_log.txt              - 完整日志
  ✓ output_comparison.wav    - 音频对比文件

═══════════════════════════════════════════════════════════
  可选: 使用自己的录音
═══════════════════════════════════════════════════════════

如果你有录音数据:
1. 准备 input_4ch.wav (4通道WAV文件)
2. 放在程序目录 (与run.bat同一文件夹)
3. 双击 run.bat

程序会自动检测并使用你的文件。

═══════════════════════════════════════════════════════════

问题: 提示 "gcc不是内部或外部命令"
解决: 
  1. 确认已安装TDM-GCC
  2. 安装时是否勾选 "Add to PATH"
  3. 重启电脑

问题: 编译错误
解决: 确保所有文件(.c 和 .h)都在同一目录

问题: 其他问题
解决: 查看详细文档
  - README_WINDOWS.md
  - INSTALLATION_GUIDE.md
